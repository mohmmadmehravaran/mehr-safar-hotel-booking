# مهر سفر — رزرو هتل (نسخه با ویرایشگر بصری زنده)

پلتفرم جستجو و رزرو آنلاین هتل با یک **ویرایشگر بصری زنده شبیه فیگما** که با
آن می‌توانید صفر تا صد ظاهر سایت را با کلیک تغییر دهید.

## امکانات ویرایشگر بصری

- **ویرایش زنده با کلیک:** روی هر عنصر (متن، تصویر، دکمه، کادر و ...) کلیک کنید
  تا انتخاب شود و از پنل کناری (Inspector) متن، رنگ، فونت، حاشیه، سایه و... را
  تغییر دهید.
- **جابجایی و تغییر اندازه:** عناصر را با دستگیره سبز بکشید و با ۸ نقطه اطراف،
  اندازه‌شان را تغییر دهید.
- **خروج / ورود سریع به حالت ویرایش:**
  - دکمه **«خروج»** در نوار بالا حالت ویرایش را می‌بندد تا مثل یک بازدیدکننده
    عادی بین صفحات بگردید.
  - دکمه شناور **«ویرایش بصری»** (پایین-چپ، فقط برای مدیر) دوباره ویرایش را
    فعال می‌کند — از هر صفحه‌ای.
  - منوی **«صفحات»** در نوار بالا برای پرش بین صفحات بدون خروج از حالت ویرایش.
- **مگنت / اسنپ (Snap):** هنگام جابجایی و تغییر اندازه، عناصر به‌صورت آهنربایی
  به لبه‌ها و مرکز عناصر مجاور و وسط صفحه می‌چسبند و خطوط راهنمای صورتی نمایش
  داده می‌شوند. با دکمه **«مگنت روشن/خاموش»** قابل کنترل است.
- **نمایش فاصله‌ها (Spacing):** با انتخاب هر عنصر، فاصلهٔ آن تا کادر والد از
  چهار طرف (چپ، راست، بالا، پایین) با خط‌چین و عدد پیکسلی نمایش داده می‌شود.

> ورود مدیر: مسیر `/#/admin` با نام کاربری `admin` و رمز `admin123`.

## اجرا به‌صورت محلی

```bash
npm install
npm run dev
```

## ساخت نسخهٔ تولید

```bash
npm run build      # خروجی تک‌فایلی در dist/index.html
```

## استقرار (Deploy)

با هر push روی شاخهٔ `main`، گردش‌کار GitHub Actions (`.github/workflows/deploy.yml`)
پروژه را ساخته و روی **GitHub Pages** منتشر می‌کند.

## تغییرات نسخه جدید — ویرایشگر بصری حرفه‌ای (فتوشاپ + کانوا)

قابلیت‌های اضافه‌شده به «ویرایش بصری»:

### مدیریت لایه‌ها (مثل فتوشاپ)
- **پنل لایه‌ها** شناور و قابل‌جابجایی (دکمه «لایه‌ها» در نوار ابزار): فهرست همه عناصر صفحه به ترتیب رو‌هم‌قرارگرفتن.
- **کشیدن‌و‌رها (Drag & Drop)** برای تغییر ترتیب لایه‌ها (رو هم / زیر هم).
- **نمایش/مخفی کردن** هر لایه با آیکون چشم 👁.
- **قفل/بازکردن قفل** لایه 🔒 (لایه قفل‌شده جابجا یا تغییر اندازه نمی‌شود).
- **تغییر نام** لایه (دابل‌کلیک یا آیکون مداد).
- **کپی** و **حذف** لایه از داخل پنل.
- دکمه‌های «یک لایه جلوتر/عقب‌تر» و «کاملاً جلو/عقب».

### امکانات طراحی (مثل کانوا)
- **کتابخانه فونت** فارسی و لاتین (وزیرمتن، لاله‌زار، امیری، نسخ، کوفی، Poppins، Playfair و…) با بارگذاری خودکار از Google Fonts.
- **چرخش عنصر (Rotation)** با اسلایدر و پیش‌فرض‌های ۰/۴۵/۹۰ درجه.
- **ترازبندی در صفحه**: چپ‌چین، وسط‌چین، راست‌چین نسبت به عرض صفحه.
- **کپی سریع** عنصر (Duplicate).

### میانبرهای کیبورد
- جهت‌نماها: جابجایی ۱ پیکسلی (با Shift: ۱۰ پیکسل)
- `Delete` / `Backspace`: حذف عنصر
- `Ctrl/Cmd + D`: کپی عنصر
- `Ctrl/Cmd + ]` و `Ctrl/Cmd + [`: جلو/عقب بردن لایه
- `L`: قفل/بازکردن قفل لایه
- `Esc`: لغو انتخاب
